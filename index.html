Project Description

This project aims to develop a comprehensive indoor navigation assistant for the visually impaired. The proposed solution will involve a smartphone app which will provide precise turn-by-turn navigation instructions through voice guidance and vibration. The app will have two ways of communicating with the user: earbuds/sounds. It will use object detection to identify if an object is inside their walking path. It will tell you what the object is by saying 10 meters, 5 meters, object left, object right. Etc. The second feature tells you what direction to go with a simple buzz. One buzz is to go left and two to go right. There will also be a feature to download specific file maps for a given area to help with the user's directions. Lastly there will be the ability to use Google maps and Google speech to text to help start navigations. Key considerations include building or integrating an indoor map database, refining obstacle detection techniques, and ensuring an intuitive, accessible user interface for the visually impaired. The ultimate goal of this project is to empower visually impaired individuals with a greater sense of independence and safety while navigating indoor environments.
