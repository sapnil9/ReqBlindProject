Project Description

This project aims to develop a comprehensive indoor navigation assistant for the visually impaired. The proposed solution will involve a smartphone app integrated with a Bluetooth-enabled walking stick. The app will provide precise turn-by-turn navigation instructions through voice guidance, while the stick will offer enhanced obstacle detection (potentially using a built-in camera) and transmit directional cues to the user. Key considerations include building or integrating an indoor map database, refining obstacle detection techniques, and ensuring an intuitive, accessible user interface for the visually impaired. The ultimate goal of this project is to empower visually impaired individuals with a greater sense of independence and safety while navigating indoor environments.
